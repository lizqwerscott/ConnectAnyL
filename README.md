[![pdm-managed](https://img.shields.io/badge/pdm-managed-blueviolet)](https://pdm.fming.dev)
</div>

## Install

首先需要安装[pdm](https://github.com/pdm-project/pdm)


安装完成后执行
```bash
pdm install
```

或者可以使用requirements.txt安装
```bash
pip install -r requirements.txt
```

## Run

```bash
pdm run start
```

如果没使用pdm安装
```bash
python src/main.py
```
